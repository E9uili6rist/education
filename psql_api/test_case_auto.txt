Заголовок: Управление БД Postgres с помощью Postman в приложении server.py .
Предусловие: Открыт Docker Desktop и запущен Docker Compose "py_to_psql" с контейнерами Postgres и server.py ;
             Открыт PyCharm и запущено приложение test_server.py .


----------------------Тест 1----------------------
Описание: Проверка получения данных из пустой таблицы .

Порядок действий: 1) Перейти в терминал из каталога "py_to_psql" ;
                  2) Выполнить команду test_server.py::test_get_data_empty .

Ожидаемый результат: "passed" .
